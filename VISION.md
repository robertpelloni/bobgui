# VISION

Bobtk (formerly Bobgui/GTK) aims to be the #1 greatest UI library of all time, delivering a robust, modern, and unparalleled development experience.

## Objectives
- **100% Feature Parity:** Achieve complete 1:1 feature parity with Qt6 (and beta Qt7 modules), JavaFX, WxWidgets, WinUI, and DearImgui.
- **Backwards Compatibility:** Maintain mostly backwards-compatible architecture with legacy GTK software, ensuring seamless transitions while drastically improving internal architecture.
- **JUCE and Ultimate++ Integration:** Intermix JUCE widgets/functions and the Ultimate++ utility toolkit with Bobtk widgets under a unified package framework, creating endless architectural paradigms and design patterns.
- **Go Port:** Transition and port all features to a comprehensively implemented, top-tier Go architecture alongside the C/C++ version.
- **Comprehensive Documentation:** Implement thorough UI documentation, labels, descriptions, and tooltips alongside high-quality manuals and help files.
